Explanation:

Inline styles are styles applied directly to HTML elements using the style attribute. They are useful for quick styles but can lead to cluttered code.

Hello, World!

This paragraph is bold and green.

This is a styled div.

Pros and Cons of inline styles:

    Pros:
  • ,Quick application of styles.
  • No need for external stylesheets for small changes.

    Cons:

  1. Not reusable.
  2. Makes HTML code harder to read and maintain.
