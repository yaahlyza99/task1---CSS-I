Explain about inline, internal and external css combination advantages and disadvantage

Inline CSS: tyles applied directly to HTML elements using the style attribute.

  • Advantages: Quick and easy for small changes.
  • Disadvantages: Not reusable; makes HTML messy.

Internal CSS: Styles defined within a style tag in the tage head section of the HTML document.

  • Advantages: Easy to manage for a single document; keeps styles separate from content.
  • Disadvantages: Not reusable across multiple pages; larger files can become cluttered.

External CSS: Styles written in a separate .css file and linked in the HTML document using a tag.

  • Advantages: Reusable across multiple pages; keeps HTML clean and manageable.
  • Disadvantages: Requires an additional HTTP request to load the CSS file.
